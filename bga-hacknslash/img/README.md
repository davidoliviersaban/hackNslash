# BGA Assets

This folder contains assets deployed to BGA Studio.

- `cards/`: card images or generated sprites.
- `tiles/`: dungeon tile images or sprites.
- `tokens/`: markers, wounds, action points, target indicators.
- `metadata/`: BGA metadata images such as box, banner, icons.

Use `scripts/prepare-bga-assets.sh` from the repository root to copy current prototype assets into this structure.
